Welcome to this Jekyll Portfolio Theme! This theme comes with a homepage "Portfolio" gallery, a blog page with a Disqus comment section, and an "about me" page. Scroll down to check out the portfolio gallery, which you can use to showcase screenshots of your designs, your photography, embeded youtube/vimeo videos, etc. Add links to your own social media accounts in the footer below. Feel free to jump into the "css/common.css" file to adjust the color scheme and fonts. Customize ANYTHING you want and make this site yours! If you'd like to contact me directly, you can reach me at miles@milesrose.net. Also, I'm currently using this theme on my personal site, if you'd like to see it in action.

Picture Portfolio

This is where you'll actually put your portfolio projects. Even though all of the pictures are different aespect ratios, they all fit neatly into a their own columns.
You can change the number of columns in the folder "css/responsive-gallery.css". I'm using Lity.js to create the light-box transition effect.
